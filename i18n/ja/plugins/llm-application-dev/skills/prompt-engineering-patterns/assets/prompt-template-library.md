## 抽出テンプレート

### 固有表現抽出
```
テキストからすべての固有表現を抽出して分類してください。

テキスト: {text}

エンティティ（JSON形式）:
{
  "persons": [],
  "organizations": [],
  "locations": [],
  "dates": []
}
```

### 構造化データ抽出
```
求人情報から構造化情報を抽出してください。

求人情報:
{posting}

抽出された情報（JSON）:
{
  "title": "",
  "company": "",
  "location": "",
  "salary_range": "",
  "requirements": [],
  "responsibilities": []
}
```

## 質問応答テンプレート

```

{context}

質問: {question}

回答:
```
